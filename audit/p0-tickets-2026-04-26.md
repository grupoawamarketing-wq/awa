### TICKET-001 — P0-6: Habilitar CSS merge e minify no admin
- **Prioridade:** P1
- **Tipo:** Config Admin Magento
- **Problema:** 36 `<link rel="stylesheet">` na home. CSS merge=0, minify=0
- **Ação:**
  1. Admin > Stores > Configuration > Advanced > Developer > CSS Settings
  2. Merge CSS Files = Yes
  3. Minify CSS Files = Yes
  4. Flush cache
- **Risco:** Merge pode quebrar CSS que depende de ordem de cascata. Testar em staging primeiro.
- **Alternativa:** Usar bundling manual (webpack) ou critical CSS inline
